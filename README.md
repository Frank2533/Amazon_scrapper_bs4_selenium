# Amazon_scrapper_bs4_selenium

This program scrapes data from amazon website.<br>
## Data includes:-<br>
<li>Product Name</li>
<li>Star rating</li>
<li>Review Rating</li>
<li>Was Price</li>
<li>Current price</li>
<li>Product Description</li>
<li>ASIN Number</li>
<li>Product URL</li>

## Output Format:-<br>
Excel (.xlsx)
